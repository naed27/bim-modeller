# BIM Modeller

## Overview
BIM Modeller is a web-based application designed for Building Information Modeling (BIM) professionals. It allows users to visualize, manage, and interact with 3D building models efficiently. The application integrates IFC (Industry Foundation Classes) support to streamline project workflows.

## Features
- View and navigate 3D BIM models.
- Load and manage IFC files.
- Interactive model inspection and selection.
- Modular UI components for efficient workflow.
- Responsive design for various screen sizes.

## Tech Stack
- **Frontend:** React / TypeScript
- **3D Rendering:** Three.js / @thatopen/ui
- **State Management:** React Context API
- **Version Control:** Git / GitHub
